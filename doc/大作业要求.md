# Python应用系统开发

## 选题要求

根据学生收集和整理资料，以及学生的兴趣，可在选题范围内选择，每人1题。班级内每人选题不重复。

## 开发语言和开发环境

1. 开发语言：Python语言
2. 开发环境：Windows 10及以上平台+Python 3.10及以上版本

## 开发要求

1. 采用数据库存储数据。
2. 实现系统基本的查询、插入、删除、修改功能，在此基础上可增加高级操作 （例如排序、统计、可视化、云部署等）。
3. 交互界面设计要直观体现系统功能选择，并可循环调用。

## 提交要求

1. 系统各部分运行时录屏视频文件（录制一个完整的视频文件，视频中需要有运行时的解说。视频必须用录屏软件录制）。
2. 期末大作业论文（包括功能框图、算法流程图、数据类型表示和定义、界面图示、各模块—函数功能简介和使用说明、系统开发总结—类型分析、算法亮点、难点和技巧等）。
3. 系统源代码
4. 以上3部分材料，需要提交一个压缩文件（包含所有数据文件和程序文件）。压缩文件命名方式为：【学号+姓名+Python程序设计大作业】。
